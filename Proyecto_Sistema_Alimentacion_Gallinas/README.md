# Sistema de Registro de Alimentación de Gallinas

## Descripción
Sistema desarrollado en C para el registro y control de la alimentación de gallinas en granjas avícolas. Permite llevar un historial completo de consumo, costos y gestión de inventario.

## Características
- ✅ Registro diario de alimentación
- ✅ Gestión de inventario de alimentos
- ✅ Cálculo de consumo promedio
- ✅ Recomendaciones nutricionales
- ✅ Búsqueda y filtrado de registros
- ✅ Reportes de costos y consumo
- ✅ Compatible con Linux

## Ejecutar
./sistema_gallinas

## Generar registros por defecto
cd Generar_registros_por_defecto
./generar_registros.sh
