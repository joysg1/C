# Sistema de Nómina en C

Un sistema de gestión de nómina desarrollado en lenguaje C que permite administrar empleados y calcular salarios.

## Características

- ✅ Gestión de empleados (agregar, listar, buscar)
- ✅ Cálculo automático de nómina
- ✅ Almacenamiento persistente en archivo de texto
- ✅ Interfaz de menú intuitiva
- ✅ Compatible con Linux (Manjaro, Ubuntu, etc.)

## Requisitos del Sistema

- Compilador GCC
- Sistema operativo Linux
- Terminal bash
